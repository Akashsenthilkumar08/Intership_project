A modern web-based Employee Management System built using HTML, CSS, JavaScript, and Firebase. This application helps organizations manage employee records, attendance tracking, and authentication through a user-friendly dashboard.

Features
Secure User Registration and Login
Firebase Authentication Integration
Employee Dashboard
Attendance Management
Real-time Data Storage with Firebase
Responsive and Modern UI Design
Employee Profile Management
Dark/Light Theme Support
QR Code-Based Attendance (if implemented)
Technologies Used
HTML5
CSS3
JavaScript (ES6)
Firebase Authentication
Firebase Firestore Database
Font Awesome Icons
